# SignupScript

The SingupScript for ZVMS.

## Statements

- `must`
- `just`
- assignment

## Operators

> by priority

1. union: `&`, `|`
2. `in`
3. algebra: `+`, `-`
4. comparison: `==`, `!=`, `>`, `>=`, `<`, `<=`
5. logic: `&&`, `||`
6. assignment: `=`

## Types

- student
- union (of student)
- int
- boolean

## IO

### Input

- `current`: union
- `new`: student

### Output

`true` or `false`
